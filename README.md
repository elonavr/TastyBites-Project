# TastyBites Web Project

A full-stack restaurant management system built with PHP and MySQL.

## Features

- Order management with Database integration.
- Dynamic order history table.
- Calorie calculation logic.

## Setup

1. Clone the repository.
2. Rename `includes/db_connect.example.php` to `includes/db_connect.php`.
3. Add your database credentials in `includes/db_connect.php`.
